# M_LOTR_Quest
# 💍 Monopoly LOTR: Traductor App web compañera para el juego físico Monopoly: El Señor de los Anillos (Edición Inglés). Traduce las cartas de misión (Quests) al español y gestiona los inventarios secretos de la Comunidad. Desarrollada con HTML, CSS y JS puro, guardando el progreso localmente. ¡Juega sin barreras de idioma!
